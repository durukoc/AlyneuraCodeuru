# AlyneuraCodeuru — site statique (français)
Ce dossier contient un site d'exemple sans informations personnelles, prêt pour hébergement gratuit (GitHub Pages, Netlify, etc.).

## Publication gratuite sur GitHub Pages
1. Créez un dépôt sur GitHub, nommez-le `votrecompte.github.io` pour un lien court.
2. Chargez `index.html` et `favicon.svg` à la racine du dépôt.
3. Dans les paramètres du dépôt, section *Pages*, activez le déploiement depuis la branche `main`.
4. Votre site sera disponible à l'adresse `https://votrecompte.github.io`.

> Remarque : L'adresse `.com` nécessite l'achat d'un nom de domaine. Cette version est gratuite et publique.
